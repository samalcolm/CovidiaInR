# CovidiaInR
Nate Silver's Covidia model ported to R

This repository contains an R function of [the spreadsheet model introduced here.](https://fivethirtyeight.com/features/coronavirus-case-counts-are-meaningless/)

```R
case = Covidia538(...)
```
```R
args(Covidia538)
```

```R
case$params
```

```R
case$output
```
